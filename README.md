- [x] Search by user

- [x] Follow / Unfollow User

- [x] See user profile
- [x] See my profile
- [x] See the full photo

- [x] Edit the post
- [x] Delete the post
- [x] See the feed
- [x] Send Message
- [x] Send private Message
- [x] See rooms
- [x] See room
- [x] Receive Message (Realtime)
